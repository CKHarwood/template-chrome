<static> utils.get(key)

Retrieves the value associated with the specified key from localStorage.

If the value is found it is parsed as JSON before being returned; otherwise undefined is returned.

Parameters:
{String} key
The key of the value to be returned.
Returns:
The parsed value associated with the specified key or undefined if none exists.
See:
JSON.parse

<static> utils.init(key, defaultValue)

Initializes the value of the specified key in localStorage.

If the value is currently undefined the specified default value will be assigned to it; otherwise it is reassigned to itself.

Parameters:
{String} key
The key whose value is to be initialized.
defaultValue
The value to be assigned to the specified key if it is currently undefined.
Returns:
The previous value associated with the specified key. This will be undefined if not previously initialized.
See:
utils.set
<static> utils.set(key, value)

Sets the value of the specified key in localStorage.

If the specified value is undefined it is assigned directly to the key; otherwise it is transformed to a JSON String.

Parameters:
{String} key
The key whose value is to be set.
value
The value to be assigned to the specified key.
Returns:
The previous value associated with the specified key or undefined if there was none.
See:
JSON.stringify
